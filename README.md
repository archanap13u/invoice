# Invoice Manager - Quick Start Guide

## 🚀 Getting Started

Simply open the application by double-clicking `index.html` or opening it in your browser:
**file:///d:/invoice/index.html**

## 📝 Creating Your First Invoice

1. Click **"New Invoice"** button
2. Fill in the form (fields marked with * are required)
3. Add multiple items using **"Add Item"** button
4. Watch the live preview on the right
5. Click **"Save Invoice"**

## ✏️ Quick Tips

- **Auto-Generated Numbers**: Invoice numbers are automatically generated in format `INV-202602-0001`
- **Live Preview**: See your invoice update in real-time as you type
- **Add Multiple Items**: Click "Add Item" to add service line items
- **Tax Calculation**: All taxes (CGST, SGST, IGST) are calculated automatically
- **Data Persistence**: All invoices are saved automatically to your browser

## 🔍 Finding Invoices

Use the search box to quickly find invoices by:
- Invoice number
- Client name
- Date
- Any text in the invoice

## 💾 Downloading Invoices

1. Click any invoice card to view
2. Click **"Download / Print"** button
3. Use browser's print dialog
4. Select "Save as PDF" to download
